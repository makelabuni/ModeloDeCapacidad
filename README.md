# ModeloDeCapacidad

Modelo para medir la capacidad de un equipo que desarrolla componentes de software.

## ¿Qué es?

Herramienta para estimar y monitorear la capacidad real de un equipo de desarrollo, considerando factores como disponibilidad, velocidad histórica y composición del equipo.
